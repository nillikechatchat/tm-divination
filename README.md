## 🌟 概述

**TM（天命）推演** 是一个遵循 [Agent Skills 开放标准](https://vortezwohl.github.io/agent/2026/03/02/AgentSkills%E5%BC%80%E6%94%BE%E6%A0%87%E5%87%86.html) 的 AI Skill。它基于四柱八字、紫微斗数、周易、黄历等中国传统国学体系，为用户提供：

- 个人命盘与先天气质分析
- 今日 / 明日 / 本周运势推演
- 今日宜忌与现代生活场景映射
- 命格适合度与发展方向建议
- A轨（传统文化表述）+ B轨（现代社会主义精神文明表述）双轨输出

所有结论均融入马克思主义哲学与积极心理学视角，**坚决反对宿命论**，强调人的主观能动性、社会实践与持续奋斗。

## 🚀 功能特性

| 特性 | 说明 |
|------|------|
| 🔮 八字排盘 | 基于公历/农历生日自动排四柱（年、月、日、时），支持三柱简化分析 |
| 📅 运势推演 | 结合当日干支、节气、十二建星与个人命盘，生成今日运势与宜忌 |
| 🧭 命格分析 | 通过十神系统与现代心理学重释，给出发展方向与职业建议 |
| 🌓 双轨输出 | A轨传统文化表述 + B轨现代精神文明表述，兼顾文化底蕴与现代价值 |
| 🛡️ 伦理合规 | 内置敏感词转换、禁止事项与安全声明，避免封建迷信与消极宿命论 |
| 🌐 跨平台 | 遵循 [Agent Skills 开放标准](https://vortezwohl.github.io/agent/2026/03/02/AgentSkills%E5%BC%80%E6%94%BE%E6%A0%87%E5%87%86.html)，可被 Claude Code、OpenAI Codex、Gemini 等兼容框架加载 |

## 📦 安装

将本仓库克隆或下载到本地，然后将 `tm-divination` 目录放入你的 Agent Skills 目录：

```bash
# Claude Code 项目级 Skill
cp -r skills/tm-divination ./.claude/skills/

# 或 Claude Code 全局 Skill
# cp -r skills/tm-divination ~/.claude/skills/
```

> 本 Skill 为**纯 AI 推理型**，无需安装外部 CLI、Python 包或 API 密钥。

## 💬 快速开始

在支持 Agent Skills 的对话中，使用以下任意话术触发：

- "帮我算一下八字 / 推演一下天命 / 看看我的命盘"
- "今天 / 明天 / 本周运势如何 / 运程怎么样"
- "分析一下我的命格 / 先天气质 / 人生格局"
- "我适合做什么工作 / 发展方向 / 职业选择"
- "今日宜忌 / 黄历查询 / 今天适合做什么"
- "根据我的出生日期分析人生阶段 / 命运走向"
- "前世今生 / 我的前世是什么 / 因果分析"

### 示例输入

```text
帮我推演一下天命，我叫李明，男，1990年5月20日下午3点出生，出生地北京。
```

### 示例输出结构

**【B轨·现代社会主义精神文明表述】**
> 李明先生，您的日主为乙木，命局火金旺相、木气偏弱……建议今日主动联系潜在合作伙伴，发挥比劫（团队协作）优势。

**【A轨·传统文化表述】**
> 乙木日主，生于巳月，官杀混杂，幸有甲木帮身……宜以印化杀或食伤制杀。

> **声明**：以上分析基于传统文化模型与哲学诠释……命运并非预定程序，而是主观能动性与客观条件在实践中的辩证统一。

## 📝 输入参数

| 参数名 | 类型 | 是否必填 | 说明 |
|--------|------|----------|------|
| `birth_date` | 字符串 | 必填 | 出生日期，优先公历（`YYYY-MM-DD`），兼容农历（如"1990年四月初五"） |
| `birth_time` | 字符串 | 选填 | 出生时辰（如"午时"、"14:30"），用于排时柱；未提供则只排三柱 |
| `name` | 字符串 | 选填 | 姓名，用于音韵文化分析与五格数理参考 |
| `gender` | 字符串 | 选填 | 性别（男/女），影响大运排法（阳男阴女顺排，阴男阳女逆排） |
| `birth_location` | 字符串 | 选填 | 出生地点，用于真太阳时校正提示 |

---

## 🎭 输出格式

所有分析结论均同时输出两轨：

- **A轨·传统文化表述**：使用典雅文言风格，引用《易经》《渊海子平》《三命通会》等经典，使用天干地支、五行、十神、卦象等传统术语。
- **B轨·现代社会主义精神文明表述**：使用马克思主义哲学概念、现代心理学概念与社会主义核心价值观进行诠释，语言平实、积极、具有建设性。

输出顺序：**先 B 轨，后 A 轨**，确保用户首先接收到现代科学语言与积极价值观的引导。

## ⚖️ 安全与伦理

本 Skill 严格遵循以下原则：

- **非宿命论**：命盘是「先天气质模型」与「人生发展倾向图谱」，绝非决定论。
- **积极重构**：所有传统「凶象」必须转化为建设性行动建议。
- **明确禁止**：不预测疾病、伤残、死亡；不提供赌博/投机/非法集资指导；不宣扬鬼神、轮回、封建迷信；不涉及政治、宗教、民族敏感关联。
- **强制声明**：每次完整推演末尾附加安全声明，强调社会实践、持续学习与艰苦奋斗。
